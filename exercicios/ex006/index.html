Principais formatações

Negrito / Destaque

Uma forma é usando a tag <b> (não semântica).

Outra forma é usando a tag <strong> (semântica).

Itálico / ênfase

Uma forma é usando a tag <i> (não semântica).

Outra forma é usando a tag <em>(semântica).

Texto marcado

Uma forma é usando a tag <mark>.

Texto grande e pequeno

Temos a tag obsoleta Big, e a não tão obsoleta small.

Texto excluído

Podemos usar a tag secreta <del>

Texto inserido

Um texto inserido posteriormente pode ser representado pela tag <ins>

Texto sobrescrito

Para coisa x20+3 use a tag <sup>

Texto subscrito

Por exemplo, H2O usa a tag <sun>
